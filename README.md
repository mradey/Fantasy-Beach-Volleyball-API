# Fantasy-Beach-Volleyball-API
An API to support a fantasy sports application. This is all for fun and learning :)

## Setup
### Start the virtual environment
**Windows**
```
python -m venv venv
venv\Scripts\activate
```

**macOs/linux**
```
python3 -m venv venv
source venv/bin/activate
```

### Install dependencies
Going to use a requirements.txt file for dependency management. To install all dependencies run the following command

```bash
pip install -r requirements.txt
```

### Run the program
**Windows** execute the batch script 
```bash 
./run.bat
```

**macOs/linux** execute the make file 
```make runDev```

## Data Sources
There are a few apis and historical data references we can integrate with. 
- [Sport Radar](https://developer.sportradar.com/volleyball/reference/beach-volleyball-overview) API
- [Beach Volleyball Devs API](https://rapidapi.com/sportdevs-sportdevs-default/api/beach-volleyball-devs) API
- [Beach Volleyball Database](http://www.bvbinfo.com/default_home.asp) Historical Data
